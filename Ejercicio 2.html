Clases del mismo nombre

Para que un elemento pueda tener una clase o un nombre para llamarlo habrá que escribir el atributo class que permite el nombrar a cualquier variable y distingirlos en CSS para poder editarlo por partes o de distintas maneras cada párrafo o cada palabra.

Ahora para que varias variables puedan tener el mismo nombre o clase pues pueden ocupar varias formas, por ejemplo pueden utilizar la variable div para separar un párrafo de otro (Me refiero a encerrar a un parrafo dentro de un DIV para tener la facilidad de editarlo en CSS) o tambien pueden agregar números a las variables p que son los encargados de escribir los párrafos, tambien los span asi poder resaltar una palabra o texto dentro de esta variable o párrafo.

Pueden tambien agregarse a los titulos pero esto no pueden cambiar los numeros, ya que cada número determina el tamaño de los textos en caso de que sean las variables h ya que estan estan destinadas a los titulos. Pero si pueden utilizar la variable SPAN para resaltar una palabra en especifico.

Antes de explicar como llamarlos en CSS , cabe recalcar que cada una de las variables que estoy usando para el diseño de este pagina esta con la clase Nombre1.

Como llamarlos en CSS

Ahora para llamar a estas variables en CSS tocara nombrarlos de distintas maneras según la comodidad del cliente o usuario

*En caso de que quiera que todo el diseño de la pagina se cambie, tanto palabras como entorno, colocará de la siguiente manera "body{}".

*En caso de que solo quiera editar los párrafos, (todos los parrafos de la pagina) solo pondrá "p {}".

*En caso de que quiera editar un párrafo especifico tocara nombrar a la variable que utilizo para que distingir ese párrafo de otros, por ejemplo, puso un div para encerrar a ese párrafo pues para llamarlo sería "div.nombredelaclase{}", ahora puso un número que lo distingue a los demas párrafos ejemplo p1 , para llamarlo sería "p1.nombredelaclase{}".

*En caso de que quiera editar un titulo colocará de la siguiente manera "h1.nombredelaclase{}".

*En caso de que quiera editar un subtitulo o una varible "h" que sobrepase el numero 1 "por que no se puede repetir un titulo principal", colocará de la siguiente manera "hnumero.nombredelaclase{}".

*En caso de que quiera editar una palabra o palabras específicas debera colocar lo siguiente "span.nombredelaclase{}".
